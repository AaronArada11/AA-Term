# AA-Term

A terminal-style personal site for collecting and sharing my profiles in one place.

My main site is [https://aaron-arada.vercel.app/](https://aaron-arada.vercel.app/).

## Installation

Clone this repository.

Install the dependencies with `npm install`.

Start the local development server with `npm run dev`.

Open [localhost:3000](http://localhost:3000) in your browser.

Enjoy!
